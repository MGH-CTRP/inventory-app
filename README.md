# 창고 재고조사 앱 - 설치 안내

## 📁 파일 구성
```
inventory-pwa/
  ├── index.html      ← 앱 본체
  ├── manifest.json   ← PWA 설정
  ├── sw.js           ← 오프라인 캐시
  ├── icon-192.png    ← 앱 아이콘
  └── icon-512.png    ← 앱 아이콘 (대형)
```

---

## 🚀 배포 방법 (Netlify - 무료, 5분)

1. https://app.netlify.com 접속 → 구글 계정으로 로그인
2. "Add new site" → "Deploy manually" 클릭
3. **이 폴더(inventory-pwa)를 드래그앤드롭**
4. 자동 배포 완료 → 주소 발급 (예: `https://amazing-name-123.netlify.app`)
5. 해당 주소를 핸드폰 크롬/사파리로 열기

---

## 📱 핸드폰에 앱 설치 방법

### 안드로이드 (크롬)
1. 앱 주소 접속
2. 주소창 오른쪽 점 3개 메뉴 → "홈 화면에 추가"

### 아이폰 (사파리)
1. 앱 주소를 **사파리**로 접속 (크롬 불가)
2. 하단 공유 버튼(□↑) → "홈 화면에 추가"

---

## 📋 사용 방법

### 1. 제품 마스터 등록 (최초 1회)
1. ⚙ 설정 버튼 → "템플릿 다운로드"
2. 엑셀 열어서 제품 정보 입력 (바코드, 제품명, 공급업체, 박스입수량)
3. 또는 수입관리 시스템에서 제품 목록 내보내기
4. "엑셀 파일 선택"으로 업로드 → 컬럼 매핑 확인 → 적용

### 2. 재고 조사
1. "스캔 시작" → 카메라 권한 허용
2. 바코드에 카메라 갖다 대기 → 자동 인식
3. 제품명 자동 표시 확인 → CTN / PCS 입력 → 추가
4. 반복

### 3. 결과 내보내기
- "엑셀 내보내기" → `재고조사_YYYY-MM-DD.xlsx` 다운로드

## ✅ 주요 기능 요약

| 기능 | 내용 |
|------|------|
| 바코드 자동 인식 | EAN-13, EAN-8, CODE-128, QR 등 |
| 직접 입력 | 카메라 불가 시 바코드 번호 수동 입력 |
| 제품명 자동 표시 | 마스터 등록 제품은 스캔 즉시 표시 |
| 박스 입수량 표시 | 예: 1 CTN = 12 PCS |
| 오프라인 작동 | 인터넷 없는 창고에서도 사용 가능 |
| 데이터 영구 저장 | 앱 꺼도 데이터 유지 (IndexedDB) |
| 엑셀 내보내기 | 바코드, 제품명, 공급업체, CTN, PCS |
